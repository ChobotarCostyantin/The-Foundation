# Фонд SCP
Фонд SCP (SCP Foundation) - це некомеритна організація, яка займається розробкою та впровадженням технологій для безпечного утримання аномальних об'єктів.

## Мета
Фонд SCP розробляє та впроваджує технології для безпечного утримання аномальних об'єктів. Наша мета полягає в використанні штучних алгоритмів та машинного навчання для виявлення аномальних властивостей об'єктів.

## Проекти
Фонд SCP розробляє наступні проекти:
* Утримання аномальних об'єктів
* Дослідження аномальних властивостей
* Безпечна моніторинг та безпека утримуваних об'єктів

## Співробітництво
Фонд SCP співпрацює з іншими організаціями для досягнення спільних мети.
